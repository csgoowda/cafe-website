# 📖 About

**Mysur Cafe** is a modern, responsive restaurant website designed to showcase authentic South Indian cuisine. The website provides an attractive interface for customers to explore the menu, learn about the café, view special dishes, and contact the restaurant.

---

# ✨ Features

- 🏠 Responsive Home Page
- 🍽️ Interactive Food Menu
- 📖 About Us Section
- 🍛 Chef's Special Dish
- ⭐ Customer Testimonials
- 📞 Contact Information
- 📱 Mobile-Friendly Navigation
- ⬆️ Back to Top Button
- 🎨 Smooth Animations
- ⚡ Fast Loading

# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/csgoowda/mysur-cafe.git
```

### Open the Project

Open the project folder and launch:

```text
index.html
```

in your favorite web browser.

No installation or server setup is required.
